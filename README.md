# Esprit-PW-2A32-2526-HappyBite
Plateforme web de nutrition intelligente offrant des recommandations personnalisées via un profil santé. Elle inclut un frigo virtuel pour proposer des recettes et réduire le gaspillage, un module de dons, un système de gamification et un forum d’échange avec la communauté.
